API

Introduction
The Semalt API allows to interact directly with the SEMALT platform, increasing the efficiency of managing large amount or complex campaigns. With the SEMALT API you can build software that manages campaigns from the customer level down to the keyword level. The API can do almost everything the SEMALT UI does, but programmatically.

Full access to our huge database

It gives developers the ability to build tools and applications that interact directly with Semalt’s huge keyword database of 3.5 billion search terms (350 million unique keywords).

The Semalt API offers 2 services: SERP and Keyword Suggestions. It can be used to either get the search engine results pages google data for the provided keywords or generate keyword suggestions.

Search engine results pages google

SERP service allows getting sites or key words positions in Google for the keywords that you request. Provided key words or sites can be localized to N individual locations, N languages and "Google" search network.

Types of companies that have benefited from the SEMALT API include:

  • Marketing/ advertising agencies
  • Search Engine Marketing (SEM) companies
  • Big brands managing a large number of accounts, with needs beyond what's possible within the UI.
